# Portfolio Website Specification

## Overview

A personal portfolio website for Mario Skaff, Senior Software Engineer. Responsive, bilingual (EN/FR), with dark mode.

---

## Sections

- **Header / Navbar**
  - Name/logo
  - Navigation links: About, Experience, Consulting Engagements, Skills, Contact
  - Language toggle (EN/FR)
  - Dark mode toggle

- **Hero**
  - Name, title, tagline

- **About**
  - Photo, bio, key skills

- **Experience**
  - Timeline or cards for roles/companies

- **Consulting Engagements**
  - Cards or timeline for client projects/engagements

- **Skills**
  - List or icons

- **Contact**
  - Email, LinkedIn, GitHub, contact form

- **Footer**
  - Copyright, social icons

---

## Features

- Responsive design (mobile/desktop)
- Dark mode toggle
- Bilingual: English (default) and French
- Clean, modern layout inspired by brittanychiang.com

---

## Tech Stack

- Framework: Next.js (React)
- Styling: Tailwind CSS
- i18n: next-i18next or next-intl
- Deployment: Vercel
